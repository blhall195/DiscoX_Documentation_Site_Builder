# Upgrading your DiscoX device software
{: .no_toc }

The DiscoX software is open to continous improvment, to keep you device running smoothly and take advantage of new features it's reccomended you keep up to date with the latest stable release.

## Connecting the DiscoX to your PC

The device has a latching button switch to control power to the device, press the button for at least 50ms to turn the device on and press it again to turn the device off.
